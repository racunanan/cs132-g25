Topic Clustering

We wanted to take a closer look at the content of each tweet, so we decided to use K-Means clustering to find topic clusters in the tweets. Prior to performing clustering, we used NLP to help simplify the tweet texts in order for the model to work.

We limit the model to only producing three clusters. This was because, after some trial runs, we determined that limiting the model to only three clusters produced more unique and easily identifiable topics.

Figure 1: Topic clustering

We have identified some sample tweets from each of the three topic clusters and included them in the figure above. We identified the three topics to be as follows:

  1. Effects of the K-12 changes on schooling
  2. Mandatory military service
  3. Criticisms and/or sarcasms about the Marcos-Duterte administration

Chi-Square Test of Independence

We use the Chi-Square test of independence to check if there is a relationship between an account's stance on Sara and the type of the account, which is either anonymous or identified (real names, official accounts, etc.) . We let our null hypothesis be that there is no relationship between the two. The test produced the following results.

  1. Chi-Square Statistic = 7.92
  2. P-Value = 0.019

For the test, we assume a confidence of 95% or an alpha value of 0.05. We can see that the P-value is less than the alpha value. Thus, we can reject the null hypothesis. However, the Chi-Square statistic is somewhat low. Because of this, we cannot say that there is a particularly strong relationship present between the account's stance on Sara and the account type.
